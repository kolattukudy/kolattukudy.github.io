Using Supervised and Unsupervised machine learning algorithams to predict stocks.

Posted on November 18, 2017 in posts • 14 min read

  1. Import historic price data
  2. Calculate technical indicator values
  3. Scale the data
  4. Use unsupervised classifier to identify trend cluster for each stocks in the scaled data
  5. Trend cluster values indicates the signal for buy or sell stock, this will act as our y - value or Target data.
  6. Technical indicators act as our X -value
  7. Scale the data again and split the scaled data to train, test data.
  8. Now use the ensable of supervised calssifiers for prediction
  9. verify the accuracy of prediction
  10. Choose the best classifier with higher accuracy to predic the return.
  11. Plot the difference between strategy return and the actual return.

#number of days
n = 10
#t training percentage
t = 0.8
#traing data end number and test data start number
split =int(t*len(df))

#shift high value to one day
df['high']=df['High'].shift(1)
#shift low value to one day
df['low']=df['Low'].shift(1)
#shift close value to 1 day
df['close']=df['Close'].shift(1)
df['RSI']=ta.RSI(np.array(df['close']), timeperiod=n)
df['SMA']= df['close'].rolling(window=n).mean()
#correlation between Moving Average and Close
df['Corr']= df['SMA'].rolling(window=n).corr(df['close'])
df['SAR']=ta.SAR(np.array(df['high']),np.array(df['low']),\
                  0.2,0.2)
df['ADX']=ta.ADX(np.array(df['high']),np.array(df['low']),\
                  np.array(df['close']), timeperiod =n)
df['Corr'][df.Corr>1]=1
df['Corr'][df.Corr<-1]=-1 
df['Return']= np.log(df['Open']/df['Open'].shift(1))
